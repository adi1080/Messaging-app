1. Open backend folder as intellij project
2. run the sql script from executable scripts in mysql database
3. import the collection from executable scripts into postman
3. change the password in properties file (according to your database)
4. open frontend folder in vscode
5. in new terminal (cmd) use commands ->
         1) npm install (make sure angular 15v , node 16v)
		 if asked also install bootstrap
		 2)npm install bootstrap
         3)npm install bootstrap-icons
6. To run the project use command -> ng s -o
